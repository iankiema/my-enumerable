## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:

- You need to have RUBY installed in your system.
- A Code Editor (Notepad++, VSCode, etc) or Windows command prompt(if you are on windows OS).

### Setup

Clone this repository to your desired folder:

```
git clone https://github.com/iankiema/my-enumerable.git

cd my-enumerable
```

### Install

```
No installation required
```

### Usage

To run the project, execute the run RUBY on the [project repository](https://github.com/iankiema/my-enumerable.git)

### Run tests

run the command ruby test_solution.rb

### Deployment

Deploy this project on any server: localhost, live host, cloud host.
